Creating new dashboard

To create a new dashboard click on Create Dashboard

Adding Metric to Dashboard

Click on left sidebar metrics button


Expand workflows by clicking on them. Search workflows in the panel search box.


Click on the metric to add it to dashboard.

Remember to click on dashboard Save button to save changes in the dashboard.

Saving dashboard

After making changes to a dashboard, make sure you save it, else changes will be discarded.
To save dashboard, click on Save button above the dashboard.


A new dashboard can be saved by anybody, but an existing dashboard can be overwritten only by its original creator.
You can also make a copy of an existing dashboard by saving it under a different name.


Provide a very descriptive name and accompanying description for it. After a dashboard is saved, it is automatically added to its creator's favorite list.

Accessing favorite dashboards

To access favorite dashboards, click on dashboard button from left sidebar.


Click on any dashboard, and it will load.


In case you dont see any dashboards there, it means you do not have any favorite dashboards yet.

Searching Dashboards

Existing dashboards can be searched by using search box available in top navigation bar.


It requires minimum 3 letters before start searching. When relavant dashboards are found, you can click on any one search result to load it.

Setting Home Page

When a dashboard is loaded, you can see Set HomePage button on top of it.


Click on that button and this dashboard will load as default home page everytime you access this application.

Set Dashboard favorite

When a dashboard is loaded, you can see Set Favorite button on top of it.


Click on that button and this dashboard will be added to your favorite dashboards list.

Send Email

When a dashboard is loaded, you can see Set Mail button on top of it.


Click on that button and this dashboard will be sent to your e-mail address as an attachment.

Dashboard Drill Down

To drill down dashboard data, click on dashboard filters button from left sidebar.


Click on any filter category, and it will expand.


Select filter value. Click again on same value to deselect it.


Search for filter values in panel search box.


Click on Apply button at the bottom of filters panel.


If required filters are not available here, check in respective metric filters.

Remember to click on dashboard Save button to permanently save selected filters.

Metric Card Rearrangement

You can rearrange metric cards by resizing them or by changing their position.
Resizing
Hover mouse on metric card border until the mouse pointer turns into a resizing pointer. Drag metric card either in horizontal or vertical direction to resize it.
Change Position
Press left click on metric card and drag it around dashboard to change its placement.

Metric Data

Metric data refers to the actual tabular data on which metrics were calculated. You can either download or explore it in tabular format.
Click on metric drop down button and click on Data option.

Explore Metric Data

To explore metric data, open metric data dialog and click on Add/Remove Columns button.


Select columns by moving them from available columns list to selected columns list. Unselect columns by moving them from selected columns list to available columns list.


Click on Get Data button to fetch data for selected columns. Once this data is fetched, it will be shown in tabular format.


You can do following with the data table.

  • Sort Columns
  • Resize Columns - double click on column header border to auto size it.
  • Pin (Free) Columns like in excel - Pin left or Pin right
  • Hide Columns

Remember to click on Apply button at the bottom of the dialog and then click on dashboard Save button to permanently save selected columns.

Exporting to CSV

To export metric data, open metric data dialog and click on Add/Remove Columns button.


Select columns by moving them from available columns list to selected columns list. Unselect columns by moving them from selected columns list to available columns list.


Click on Export CSV button to fetch data for selected columns. Once this data is fetched, Download CSV button will become active. Click on Download CSV button to download metric data.


Remember to click on Apply button at the bottom of the dialog and then click on dashboard Save button to permanently save selected columns.

Metric Drill Down

Click on metric drop down button and click on Filters option to open filters dialog.

Click on any filter category, and it will expand.

Select filter value. Click again on same value to deselect it.

Search for filter values in panel search box.

Click on Apply button at the bottom of filters dialog.

If required filters are not available here, check in dashboard filters.

Remember to click on dashboard Save button to permanently save selected filters.

Measures

Measures are the measurable quantities to evaluate metrics. Each metric may have multiple measures based on how they are calculated. Below image shows measures for Incident Volume.

Measure Goal

Measure goal is used to monitor health status. You can set goal on some scalar value measure.
Click on metric drop down button and click on Measures option to open Measures dialog.

Enter desired values for goal.

When a measure meets its goal it is distinctly recognizable on metric card. Metric card also changes its appearance based on goal that was met.



Remember to click on Apply button at the bottom of the dialog and then click on dashboard Save button to permanently save thresholds.

Disabling Measures

At times you might not want to see certain measures on a dashboard. To remove measures from a metric, simply disable it.
Click on metric drop down button and click on Measures option to open Measures dialog.

Click on Disable Measure button to remove it from dashboard.

Remember to click on Apply button at the bottom of the dialog and then click on dashboard Save button to permanently disable measures.

Measure Details

Measure documentation provides detailed information on a measure as how was it calcualted (formula). Its description, unit of measurement etc...
Click on metric drop down button and click on Measures option to open Measures dialog.

Click on Click Here for details link at the bottom of measure to expand details table.

Metric Details

Click on metric drop down button and click on Metric Details option to open Metric Details dialog.


Removing metric from Dashboard

Click on metric drop down button and click on Remove option to remove metric from dashboard.

Workflow

Workflows are logical containers for metrics. Every metric is part of at least one workflow. Workflows help us to organize metrics in subject area wise categorizations.
